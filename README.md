# What I've learned developing this app 

- Asynchronous tasks.
- Understand async and await.
- Futures and how to work with them.
- Networking with the Dart http package.
- Working with APIs and JSON in Flutter
- Navigation and Routes in Flutter and passing data between screens.
- Handling exceptions in Dart using try/catch/throw.
- Deep Understanding to Stateful Widgets lifecycle.
- Using Geolocator package to get live location data for both iOS and Android.
- Alert Dialogues and TextField Widgets.
